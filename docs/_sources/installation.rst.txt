Requirements
~~~~~~~~~~~~~~
This project requires at least Python 3.9 with support for up to Python 3.11+, and has official support for Ubuntu/Debian/Windows.

Required python packages for core python server:

* SQLAlchemy~=1.4.31
* SqlAlchemy-Utils~=0.38.2
* passlib~=1.7.4
* bcrypt~=3.2.0
* pydantic~=1.9.0
* fastapi~=0.73.0
* fastapi-utils~=0.2.1
* uvicorn~=0.17.5
* python-multipart~=0.0.5
* mariadb~=1.0.9
* Jinja2~=3.0.3
* PyJWT~=2.3.0
* starlette~=0.17.1
* pdfkit~=1.0.0
* requests~=2.27.1
* PyMySQL~=1.0.2

Required python packages for documentation wiki generation:

* furo==2022.2.23
* sphinx==4.4.0

Other required packages:

* Wkhtmltopdf library for Windows Installations (included with project software package archive)

~~~~~~~~~~~~~~~~~~~~~~~

Please refer to the "Setup and Installation Guide" PDF document available to you.
