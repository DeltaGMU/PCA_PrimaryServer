Student Interface#

This module contains the functions that interface with the database server to handle the processing of student records. Any code related to the handling of student records that require creating, reading, updating, or deleting data from the database server must use this interface module.

async server.lib.database_controllers.student_interface.check_student_has_records(student_id: str, session: Optional[sqlalchemy.orm.session.Session] = None) → bool#

This utility method is used to check if a student record has any associated student care service records.

Parameters

  • student_id (str, required) – The ID of the student.

  • session (Session, optional) – The database session used to retrieve student care records.

Returns

True if the student has associated student care service records.

Return type

bool

Raises

HTTPException – If the provided student ID is invalid.

async server.lib.database_controllers.student_interface.create_student(pyd_student: server.lib.data_models.student.PydanticStudentRegistration, session: Optional[sqlalchemy.orm.session.Session] = None) → Dict[str, any]#

This method creates a new student record with all associated student information such as parent contact information and student grade level, and inserts the records into the database. When a new student record is created, an email is automatically sent to the primary parent email notifying them that their child’s account has been set up.

Parameters

  • pyd_student (PydanticStudentRegistration, required) – The set of information required to register a new student record, represented by the PydanticStudentRegistration pydantic class.

  • session (Session, optional) – The database session used to insert the student information into the database.

Returns

A JSON-Compatible dictionary containing the newly created student record information.

Return type

Dict[str, any]

Raises

HTTPException – If any of the provided parameters are invalid, or a database error occurred during student creation.

async server.lib.database_controllers.student_interface.get_student_by_id(student_id: str, session: Optional[sqlalchemy.orm.session.Session] = None) → server.lib.data_models.student.Student#

This utility method is used to retrieve a student record from the provided student ID.

Parameters

  • student_id (str, required) – The ID of the student.

  • session (Session, optional) – The database session used to retrieve the student record.

Returns

The student record retrieved from the database.

Return type

Student

Raises

HTTPException – If the student ID is invalid, or the student record does not exist.

async server.lib.database_controllers.student_interface.get_student_contact_info(student_id: str, session: Optional[sqlalchemy.orm.session.Session] = None) → server.lib.data_models.student_contact_info.StudentContactInfo#

This method is used to retrieve a student record’s parent contact information.

Parameters

  • student_id (str, required) – The ID of the student.

  • session (Session, optional) – The database session used to retrieve the student’s parent contact information.

Returns

The student’s parent contact information associated with the provided employee ID.

Return type

StudentContactInfo

Raises

RuntimeError – If the student ID is null or if there is no parent contact information associated with the student record.

async server.lib.database_controllers.student_interface.get_student_grade(student: server.lib.data_models.student.Student, session: Optional[sqlalchemy.orm.session.Session] = None) → server.lib.data_models.student_grade.StudentGrade#

This method is used to retrieve the grade level of the provided student record.

Parameters

  • student (Student) – The student record used to retrieve the grade level of the provided student record.

  • session (Session, optional) – The database session used to retrieve the student grade level.

Returns

The student grade level of the provided student record.

Return type

StudentGrade

Raises

RuntimeError – If the student record is null, or the student grade level is invalid or doesn’t exist in the database.

async server.lib.database_controllers.student_interface.remove_students(student_ids: server.lib.data_models.student.PydanticStudentsRemoval | str, session: Optional[sqlalchemy.orm.session.Session] = None) → List[server.lib.data_models.student.Student]#

This method accepts one or more student IDs and deletes the corresponding student records from the database. Please note that student records that have associated student care service records cannot be deleted unless all student care service records are removed.

Parameters

  • student_ids (PydanticStudentsRemoval | str, required) – A single student ID or a list of student IDs to delete the corresponding student record(s).

  • session (Session, optional) – The database session used to delete one or more student records.

Returns

The list of student records that have been deleted from the database.

Return type

List[Student]

Raises

HTTPException – If any of the provided parameters are invalid, or the student has associated student care service records.

async server.lib.database_controllers.student_interface.update_student(student_id: str, pyd_student_update: server.lib.data_models.student.PydanticStudentUpdate, session: Optional[sqlalchemy.orm.session.Session] = None) → server.lib.data_models.student.Student#

This method updates a single student record with updated student information. Upon successful submission of the student record, an email is sent to the student’s parents notifying them that their child’s account has been updated.

Parameters

  • student_id (str, required) – The ID of the student.

  • pyd_student_update (PydanticStudentUpdate, required) – The updated student information that should be used.

  • session (Session, optional) – The database session used to update the student record.

Returns

The student record that has been updated.

Return type

Employee

async server.lib.database_controllers.student_interface.update_students(student_updates: Dict[str, server.lib.data_models.student.PydanticStudentUpdate], session: Optional[sqlalchemy.orm.session.Session] = None) → List[server.lib.data_models.student.Student]#

This method updates one or more student records with updated student information provided in the form of a dictionary consisting of student ID keys and student update information values. Upon successful submission of a student record, an email is sent to the student’s parents notifying them that their child’s account has been updated.

Parameters

  • student_updates (Dict[str, PydanticStudentUpdate]) – A dictionary containing student update information values paired with student ID keys.

  • session (Session, optional) – The database session used to update one or more student records.

Returns

A list of the student records that have been updated in the database.

Return type

List[Student]

Raises

HTTPException – If any of the provided parameters are invalid.
